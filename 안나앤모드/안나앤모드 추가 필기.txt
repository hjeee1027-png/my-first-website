<line-height>
:줄간격
->img태그는 글자의 속성을 가지고 있음
->line-height의 값은 자식에게 상속
->부모에게 주면 글자속성을 가지고 있는 img에게 상속되고
   상속으로 img의 높이가 늘어나고 div태그는 블록요소라서
   자식만큼 높이 차지함

<img가 박스보다 크거나 박스보다 작은경우>
->img에게 width : 100% 줘야 함
->이미지가 블록요소 만큼 차지하도록 퍼센트를 준다

<margin의 특징>
1)margin은 공용공간이 있다
위콘텐츠의 마진과 아래 콘텐츠의 마진이 겹치는 경우
->공용공간의 길이보다 더 긴 값을 줘야만 마진이 적용
->예제 예시 공용공간 80px + 50px => 130px

2)margin의 작성순서

margin: 0 auto
margin-top : 값

->이순서를 바꾸면 margin-top이 적용이 안됨
->top을 먼저 쓰면 값을 주고 그이후 margin : 0 auto를
   쓰면서 상하값이 0이 됨
   즉, margin-top값이 0이 됨
